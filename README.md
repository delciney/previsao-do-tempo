# Previsão do Tempo

Aplicação para consulta de previsão do tempo utilizando a API da [HG Brasil Weather](https://hgbrasil.com/status/weather).

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Variáveis de ambiente

Crie um arquivo `.env.local` na raiz do projeto com:

```
chave_hgbrasil_weather=SUA_CHAVE_AQUI
```

A chave pode ser obtida em [hgbrasil.com](https://console.hgbrasil.com/).

## Como executar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
